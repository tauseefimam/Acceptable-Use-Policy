# Acceptable Use Policy — FluxPay

## Overview
This Acceptable Use Policy (AUP) was developed for FluxPay, a fictional 
Pakistani fintech company operating under State Bank of Pakistan (SBP) 
regulatory oversight. It governs the appropriate use of FluxPay's 
information assets, technology resources, and systems by all authorized 
users.

## Context
FluxPay is a fictional company created as part of a self-directed 
60-day GRC portfolio project. The company profile:
- 2,400 employees across Karachi HQ and regional offices
- Processes customer payment data and KYC identity records
- Hybrid work model with remote access to financial systems
- Regulated by the State Bank of Pakistan

## Framework Alignment
This policy aligns with:
- ISO 27001:2022 Annex A controls — particularly A.5.10 
  (Acceptable Use), A.6.2 (Terms of Employment), A.8.1 
  (User Endpoint Devices)
- NIST CSF 2.0 Protect function
- State Bank of Pakistan cybersecurity framework requirements
